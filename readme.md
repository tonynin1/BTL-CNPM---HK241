Đây là website phục vụ việc in ấn cho sinh viên Đại Học Bách Khoa TPHCM.

# Cách để chạy website
# Yêu cầu: Tải phiên bản NodeJs từ 20 trở đi

## B1. Chạy Frontend: 
    - Đầu tiên hãy mở terminal lên, chọn chế độ chia đôi terminal.
    - Nhảy vào folder frontend bằng cách nhập 'cd frontend' và nhấn phím enter.
    - Tải các modules cần thiết để thiết lập chương trình bằng cách nhập 'npm i' hoặc 'npm install'. Nếu gặp lỗi conflicts version, hãy nhập vào 'npm i --force' hoặc 'npm install --force'
    - Sau khi thiết lập đầy đủ chương trình thì nhập vào 'npm run dev' và nhấn phím Enter để chạy code. Một cửa sổ mới xuất hiện và ra được phần giao diện của website.

## B2. Chạy Backend:
    - Nhảy vào folder cnpm-be bằng cách nhập 'cd cnpm-be' và nhấn phím enter.
    - Tải các modules cần thiết như bước 3 ở trên.
    - Chạy chương trình bằng cách nhập vào 'npm run start:dev' và nhấn phím enter.
